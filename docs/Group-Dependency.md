# Group-Dependency

## SYNOPSIS
Group dependencies by module name.

## SYNTAX

```
Group-Dependency [-Module] <ModuleSpecification> [<CommonParameters>]
```

## DESCRIPTION
Deduplicates dependencies and use the highest specified version.
